## 끝말잇기(Word-chain)


### 게임방식
1. 플레이 인원을 정한다.
2. 닉네임을 정한다.
3. 매 턴 마다 랜덤으로 순번이 정해진다.


### 코드리뷰
* 한 가지 함수에 많은 기능을 담당한다.
하나의 함수가 여러 역할을 한다는 것은, 하나의 기능이 여러 기능에 의존성을 가지고 있어 한 개를 수정하면 모두가 수정해야 한다.
또한, 코드를 읽을 때 이해하기가 힘들어진다.
* 함수를 작은 단위로 나눠서 만들기 (5줄 넘지 않게 짜보기)
* 전역변수 줄이기
* javascript가 html에 의존성이 높다.
* if 문 줄이기
* reset 함수 분할하기
* for 문 분할하기
* 플레이 부족 시 실행되는 함수 분할하기
* camelCase 사용하기
* if 문은 '삼 항 연산자'에 대해 공부해보기
* 자바스크립트를 이해하고 작성했다고 보기 어려운 코드 다.
* 구조가 없다.
